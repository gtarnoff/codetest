# How to run

1. Clone the repo
2. Run `npm install`
3. Run `npm start`, this will use [Grunt.js][00] to fire up the server on [http://localhost:8000](http://localhost:8000)
4. Run tests with `npm test`, which calls the [mocha][01] command

[00]: http://gruntjs.com/
[01]: https://mochajs.org/
